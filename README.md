## Getting started

### Dependencies

- [VCPKG](https://github.com/microsoft/vcpkg)
- [Cmake](https://cmake.org/)

## Build

```
./vcpkg install assimp bullet3 enet freetype glew libsndfile sdl2 openal-soft
mkdir build
cd build
cmake ..
cmake --build build --config Release
```
